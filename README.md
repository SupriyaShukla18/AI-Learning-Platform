## 🚀 About the Project  

This is an interactive AI-based learning platform designed to improve student productivity and learning.

### 🎯 Students can:
- 🧠 Learn concepts through quizzes  
- 📚 Join study rooms  
- ⏱️ Use Pomodoro timer  
- 🎧 Listen to focus music  
- 📝 Manage daily tasks  
- 📊 Track performance  

## ✨ Features  

### 🧠 AI Learning Quests  
- MCQ-based challenges  
- XP & streak system  

### 📚 Study Room  
- Create/join sessions  
- Task manager  
- Focus timer  
- AI assistant  

### 🎵 Focus Mode  
- Lofi / rain sounds  
- Distraction-free UI  

### 🎨 Modern UI  
- Glassmorphism design  
- Light/Dark mode  
- Smooth animations  

### 📊 Productivity Tools  
- Task tracking  
- Progress analytics
